**Intelligent Content Orchestration for Live Production Environments**

Built for houses of worship, broadcast studios, and live event producers, FlowSync resolves the most persistent friction points in media presentation: video repetition, screen mapping inconsistencies, and release note fragmentation. ## 🚀 Key Features

### 🎬 Adaptive Video Cycle Detection (AVCD)
- **What it solves:** ProPresenter’s default behavior of repeating video loops until manually stopped.
- **How it works:** FlowSync analyzes video duration, scheduled event length, and your historical usage patterns to set intelligent stop points. If a 30-second background loop would repeat 12 times during a worship set, FlowSync truncates it to 3 repeats and fades to a complementary static frame.
- **Real-world benefit:** No more awkward moments where a looping waterfall becomes a hypnotic distraction.

### 🖥️ Stage Display Synchronization Protocol
- **What it solves:** ProPresenter not showing on screen (cascade monitor, secondary display, or projection mapping issues).
- **How it works:** FlowSync maintains a persistent handshake with all output targets. If a display goes offline or a resolution shifts, FlowSync rebuilds the mapping automatically—no manual reconfiguration required.
- **Real-world benefit:** You can hot-plug a projector mid-service without losing your stage display.

### 📜 Contextual Release Notes Engine
- **What it solves:** Overwhelming generic release notes that don’t tell you what *your* installation needs.
- **How it works:** FlowSync compares your installed modules (background packs, stage display templates, audio routing) against each update’s changelog, highlighting only the sections that affect you.
- **Real-world benefit:** No more reading 40-page PDFs to find the one bug fix that matters to your display configuration.

### 🌐 Multilingual Presentation Layer
- Supports 14 languages natively, including dynamic RTL for Hebrew and Arabic scripts.
- Live translation of on-screen text overlays without affecting the original slide content.

### 📱 Responsive Media Dashboard
- Manage backgrounds, video sources, and screen layouts from any iOS, Android, or web browser.
- Full drag-and-drop timeline editing with haptic feedback on mobile devices.

## 🧱 System Architecture

FlowSync operates as a **modular overlay**, not a replacement. It sits between ProPresenter and your output hardware, intercepting commands and injecting corrections at three layers:

1. **Input Layer** – Reads ProPresenter’s event queue and media library metadata.
2. **Logic Layer** – Applies AVCD algorithms, screen mapping rules, and release note parsing.
3. **Output Layer** – Manages HDMI/NDI/StreamDeck outputs with persistent connection checks.

All data processing occurs locally. No cloud dependency, no third-party telemetry.

## 🛠️ Getting Started

### Prerequisites
- ProPresenter 7.8 or later (2026 compatible)
- macOS Ventura+ or Windows 11
- At least one secondary display (for stage display synchronization)

### Quick Configuration Walkthrough

**Step 1: Enable Adaptive Video Cycle Detection**
- Open FlowSync Preferences → Media Control.
- Toggle **AVCD Mode** to “Intelligent.”
- Set your preferred maximum loop count (default: 3).

**Step 2: Activate Stage Display Mirror**
- Navigate to Output → Display Integrity.
- Click **Auto-Detect Screens**.
- FlowSync will map all outputs and create a fallback profile.

**Step 3: Configure Release Intelligence**
- Go to Updates → Release Notes.
- Select your installed ProPresenter modules.
- FlowSync will generate a personalized changelog summary.
